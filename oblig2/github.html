Git/GitHub

GitHub er en nettbasert versjonskontroll og samarbeidsplattform for programutviklere. GitHub, som da leveres som en forretningsmodell for SaaS, ble startet i 2008 med Git som grunnlag.

Git er et "open source"-kodehåndteringssystem laget av Linus Torvalds, og hadde som mål å gjøre programvare raskere. Systemet brukes til å lagre kildekoden for et prosjekt, for spore hele historikken, og alle endringer som er blitt gjort i koden. Git tillater dermed utviklere å samarbeide mer effektivt på store prosjekter. Dette gjøres ved hjelp av verktøy som kan administrere mulige motstridende endringer fra flere utviklere. GitHub tillater utviklere å endre, tilpasse og forbedre programvare på sine offentlige oppbevaringssteder, men ønsker man en privat oppbevaringsplass (repositorier), kan det bli tilbudt betalte planer.

GitHub åpner for muligheten til sosial koding ved å gi et webgrensesnitt til Git-kodelageret, og styringsverktøyene slik at en gruppe kan samarbeide. GitHub blir da på mange måter et sosialt nettverk for programutviklere. Medlemmene kan følge hverandre, motta oppdateringer på bestemte prosjekter og rangere hverandres arbeid.

Det tillates også for utviklere å gjøre endringer uten å påvirke den opprinnelige koden. Hvis utvikleren dermed ønsker å dele endringene, kan han/hun sende en "trekkforespørsel" til eieren av det opprinnelige lageret. Hvis eieren dermed finner ut at han/hun ønsker å trekke endringene inn i original koden, så kan man fusjonere endringene med det opprinnelige lagringsstedet.

Det er mange gode fordeler med at utviklere i senere tid har valg å bruke GitHub. Det åpner mulighetene for enklere samarbeid, og gjør det enklere å holde orden på større prosjekter hvor flere utviklere jobber på samme kode. En annen fordel er at GitHub gjør det mulig å få innspill fra andre, siden alt bli lagret på et offentlig lager.

Min erfaring med GitHub har vært bra. Det var kanskje vanskelig å forstå med en gang, men når du satt deg inn i det var det ren magi å forholde seg til. GitHub er noe jeg tror jeg kommer til å bruke svært mye.
